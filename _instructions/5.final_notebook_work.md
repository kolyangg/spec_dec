## Step 1: Collect Benchmark Output Blocks

Run from the project root:

```bash
cd /path/to/hw3_spec_dec
```

List available benchmark outputs:

```bash
find benchmarks/results -maxdepth 1 -type f | sort
```

Expected files from step 4 are usually named like:

```text
benchmarks/results/baseline_bf16.txt
benchmarks/results/spec_bf16_n1.txt
benchmarks/results/spec_bf16_n2.txt
benchmarks/results/fp8.txt
benchmarks/results/fp8_spec_n1.txt
benchmarks/results/fp8_spec_n2.txt
benchmarks/results/*.json
```

Print all benchmark text summaries:

```bash
for f in benchmarks/results/*.txt; do
  echo "===== $f ====="
  grep -A80 "Serving Benchmark Result" "$f" || true
done
```

Print only the key throughput and latency lines:

```bash
for f in benchmarks/results/*.txt; do
  echo "===== $f ====="
  grep -E "Output token throughput|Total token throughput|Mean TTFT|Mean TPOT|Request throughput|Benchmark duration" "$f" || true
done
```

Choose the final speculative runs:

- BF16 speculative: highest output token throughput that also has reasonable TPOT and acceptance behavior.
- FP8 + speculative: tune independently; do not assume the same draft-token count as BF16.
